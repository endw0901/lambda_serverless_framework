- install dependencies (node , aws cli)

- install serverless framework
https://www.serverless.com/

```
// install
sudo npm install -g serverless

// create new serverless project
serverless

serverless config credentials --provider aws --key xxxxxx --secret xxxxxxxx

// add profile? (credentialsにprofileがあれば不要？ serverless-admin
??



// serverless -> slsに省略
sls create --template google-python --path hello-world-gpython
ls
cd hello-world-gpython
 
sls
// -  AWS - Node.js - Starter -> project name : aws-node-project -> deploy:yes -> 
ls
// aws-node-project, hello-world-gpython
cd aws-node-project
code .


// go to AWS管理画面 AWS Lambda -> Functions => function name: aws-node-project-dev-hello
// serverless.yamlのfunctionに、ファイル名＋メソッド名でたどれる
// code欄 => handler.js -> run test (event name: my test)

```

## code 修正・deploy

```
code .

// 一部のみdeploy => 速い
sls deploy function -f hello
```

## invoke 
- AWS管理画面でのtestをconsole上で実行し console上にログを出せる

```
sls invoke -f hello
```
